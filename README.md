## 📱 产品介绍

**小伴学**是一款创新的AI伴读应用，通过虚拟人视频陪伴、智能专注度监控、游戏化任务系统，帮助小学生培养良好的学习习惯，让家长即使不在身边也能了解孩子的学习状态。

### 🎯 核心价值

- **AI视频陪伴**: 虚拟人实时陪伴孩子学习，提供互动和鼓励
- **专注度监控**: AI智能识别孩子学习状态，分心时温馨提醒
- **特工任务系统**: 将作业变成有趣的特工任务，激发学习动力
- **家长远程监管**: 实时查看学习进度和专注度，制定学习计划

## 🖼️ 界面预览

### 主要页面

| 页面 | 功能描述 |
|------|---------|
| **首页** | 今日任务概览、专注时长统计、快捷开始学习、AI小伴问候 |
| **任务看板** | 特工风格任务列表、进度追踪、积分奖励 |
| **视频陪伴** | 虚拟人视频通话、实时专注度显示、学习计时 |

| **听写助手** | AI语音朗读、速度调节、拍照批改 |
| **AI老师** | 作业解读、知识点讲解、学习建议、每日总结 |
| **成就中心** | 等级系统、勋章展示、积分统计 |
| **家长端** | 实时监控、学习计划管理、周报告 |

## 🚀 快速开始

### 查看原型

1. 使用浏览器打开 `src/index.html` 文件
2. 点击顶部导航按钮切换不同页面
3. 也可以通过底部标签栏进行页面导航

```bash
# 使用本地服务器打开（推荐）
cd src
python -m http.server 8080
# 然后访问 http://localhost:8080

# 或者直接用浏览器打开
open src/index.html
```

## 🎨 设计规范

### 色彩系统

| 色彩 | 用途 | 色值 |
|-----|------|------|
| 🟠 活力橙 | 主色调，代表温暖与陪伴 | `#FF6B35` |
| 🔵 智慧青 | 辅助色，代表专注与智能 | `#4ECDC4` |
| 🟡 成就金 | 点缀色，代表成就与奖励 | `#FFD93D` |
| 🟢 成功绿 | 完成状态 | `#10B981` |
| 🔴 警示红 | 紧急任务 | `#EF4444` |

### 设计原则

- **温暖**: 柔和的色彩和圆角设计，传递陪伴感
- **活力**: 明快的交互动画，激发学习兴趣
- **清晰**: 简洁的界面层次，降低认知负担
- **安全**: 大圆角和柔和阴影，营造安全感

## 🔧 技术架构

### 前端技术

- **框架**: HTML5 + CSS3 + JavaScript
- **样式**: 自定义CSS设计系统 + CSS Variables
- **图标**: Font Awesome 6.5
- **动画**: CSS Animations + Transitions

### AI能力（规划中）

- **视觉AI**: 人脸检测、专注度分析
- **语音AI**: ASR语音识别、TTS语音合成
- **NLP**: 对话理解、情感分析
- **虚拟人**: Live2D/3D Avatar渲染
